Khem Raj Pradhan Personal Brand Website

OFFICIAL DOMAIN
khemrajpradhan.com.np

PROFESSIONAL TITLE
Independent Systems Researcher and Life Architect

DESCRIPTION
This is a high-performance, single-page static website designed to function as a professional research laboratory and service portal. The site is built to reflect the Khem Raj Pradhan brand identity, focusing on the deconstruction of failing social systems and the architecture of regenerative alternatives. 

The design follows Digital Vastu principles, utilizing a charcoal, gold, and terracotta palette to signal authority, wisdom, and vitality.

CORE FEATURES
1. Single-page scroll-story architecture.
2. Dynamic SVG geometric background representing systemic connections.
3. Multilens interactive interface for Space, Earth, Nature, Human, and Tech sciences.
4. Categorized Transformation Records for Personal, Business, and NGO impact.
5. Sincerity Gate integration with a secure QR payment modal.
6. WhatsApp direct-contact floating interface.
7. SEO optimization for global search visibility.

TECHNICAL STACK
1. Hosting: GitHub Pages.
2. DNS and SSL: Cloudflare.
3. Coding: HTML5, CSS3, Vanilla JavaScript.
4. Email: Zoho Mail (email@khemrajpradhan.com.np).

FILE DIRECTORY
1. index.html: Contains the structure, story, and SEO metadata.
2. style.css: Contains the brand colors, layout rules, and animations.
3. script.js: Contains the service availability logic and UI interactions.
4. CNAME: Maps the GitHub server to khemrajpradhan.com.np.
5. robots.txt and sitemap.xml: Optimize search engine crawling.
6. favicon.ico: The browser tab icon.

MAINTENANCE AND UPDATES

1. UPDATING SERVICE STATUS
To change the message about your availability (e.g., Now accepting 3 clients), open script.js and edit the text inside the quotes on the first line: const currentAvailability = YOUR MESSAGE HERE.

2. ADDING NEW REVIEWS
Open index.html and scroll to the section with the id reviews. Find a block labeled review-card. Copy that block, change the image source, name, and text, then paste it below the existing ones.

3. UPDATING QR CODES
If you change your bank or eSewa details, upload the new images with the exact names esewa-qr.png and nabil-qr.PNG to the root folder. This will automatically update them on the website.

4. CONTACT DETAILS
To update your email or physical location, go to the footer section at the bottom of index.html and edit the text inside the p tags.
